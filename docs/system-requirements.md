# 시스템 요구사항 — Smart Soy Sauce Factory

**우선순위**: R = 필수(Required), O = 선택(Optional)

---

## 1. 관리자·인증

### S-01 관리자 설정 및 로그인 기능
- **우선순위**: R
- **설명**:
  - 최초 로그인 시 비밀번호를 설정할 수 있다.
  - 이후 설정된 비밀번호로 관리자 모드에 로그인할 수 있다.

### S-02 작업자 카드 발급
- **우선순위**: R
- **설명**: 관리자는 작업자에게 ID 카드를 발급할 수 있다.
- **ID 카드 저장 데이터**:
  - 작업자 ID
  - 작업자 이름

### S-03 작업자 인증
- **우선순위**: R
- **설명**:
  - 등록된 작업자 여부를 ID 카드로 확인할 수 있다.
  - 작업장 출입 시 체크한다.

---

## 2. 작업 로그·레포트

### S-04 작업 로그
- **우선순위**: R
- **설명**: 입고 및 분류 과정에서 발생하는 작업 이벤트마다 기록한다.
- **기록 항목**:
  - 작업자별 출입 로그
  - 재고 히스토리
  - 작업 시작 및 종료 일시
  - 작업 결과
  - 작업 단계별 로그
  - 물품 인식 로그
  - 카운팅 로그

### S-05 작업 로그 조회
- **우선순위**: R
- **설명**: 작업 로그를 아래 조건에 따라 조회할 수 있다.
- **조회 조건**: 기간별, 작업자별, 물품별

### S-06 관리 레포트
- **우선순위**: O
- **설명**: 입출고 내역을 조건별로 분석·시각화하여 레포트를 생성할 수 있다.
- **조건**: 기간별, 물품별

### S-07 레포트 내보내기
- **우선순위**: O
- **설명**: 분석한 레포트를 원하는 형식으로 내보내기(Export)할 수 있다.
- **형식**: PDF, Excel, CSV

---

## 3. 입고 정보

### S-08 입고정보 관리
- **우선순위**: R
- **설명**: 물품 발주 시 주문정보를 관리한다.
- **주문 정보**:
  - 입고번호
  - 주문할 물품 코드
  - 주문할 물품 수량
- **입고 확인**:
  - 주문 고유번호가 담긴 QR 코드 사용
  - 주문 고유번호 대조로 해당 발주 주문 여부 확인
- **입고 완료 후 관리 정보**:
  - 입고번호
  - 물품 정보(물품명, 용량, 유통기한)

---

## 4. 자동 분류 공정 — 초기화·실행

### S-09 자동 분류 공정 초기화
- **우선순위**: R
- **설명**: 자동 분류 장치 전원 투입 시 최초 1회 다음을 실행한다.
  - 자동 분류 공정의 모든 부품을 가동해 정상 작동 여부 테스트
  - 최종적으로 공정 가동 상태 여부 확인

### S-10 자동 분류 공정 실행
- **우선순위**: R
- **설명**:
  - 작업자는 등록된 입고 정보를 기반으로 자동 분류 공정을 실행할 수 있다.
  - 자동 분류 공정 진행 중에는 다른 입고 정보를 추가할 수 없다.

---

## 5. 자동 분류 공정 — 물품 인식·분류

### S-11 물품 인식
- **우선순위**: R
- **설명**: 자동 분류 공정 시작 시 물품 정보를 자동 인식한다.
  - 자동 분류 장치가 입고 물품을 이동시킨다.
  - 차례대로 물품 QR 코드를 인식한다.
  - QR 코드에서 물품 상세 정보를 취득한다.
- **인식 정보**: 물품명, 브랜드(샘표, 몽고, 삼양, 오뚜기), 종류(진간장, 국간장, 양조간장, 게간장), 용량, 유통기한
- 위 정보를 바탕으로 물품 분류 기능을 실행한다.

### S-12 물품 분류
- **우선순위**: R
- **설명**: 인식한 물품의 용량에 따라 다음과 같이 분류한다.
  - 1L
  - 2L
  - 미분류

### S-13 미분류 판정
- **우선순위**: R
- **설명**: 다음 상태에서 물품을 미분류로 처리한다.
  - QR 코드가 없거나 미인식
  - QR은 인식되었으나 오분류된 경우

### S-14 미분류 물품 재분류
- **우선순위**: R
- **설명**: 작업자가 미분류된 물품을 다시 분류 대상으로 설정하여, 자동 분류 공정 종료 후 미분류 물품만 대상으로 별도 재분류를 수행할 수 있다.
  - **정상 분류 시**: 기존 '미분류' 수량은 차감되고, '정상 분류' 수량으로 반영된다.
  - **비정상 분류 시**: 미분류 수량은 유지된다.

### S-15 실시간 분류 오류 감지
- **우선순위**: R
- **설명**:
  - 시스템 판정 결과와 실제 분류 경로가 다를 경우 즉시 작업자에게 알림을 보낸다.
  - 해당 물품은 분류 미처리 상태로 남는다.

---

## 6. 자동 분류 공정 — 일시정지·검수·알림

### S-16 검수 확인
- **우선순위**: R
- **설명**:
  - 물품 분류 작업이 끝나면 자동 분류 장치가 자동으로 일시정지된다.
  - 작업자가 이 시점에 분류 결과를 검수한다.
  - 검수 완료 시 관리자에게 현황보고서를 보내고 자동 분류 공정을 종료한다.

### S-17 수동 일시 정지
- **우선순위**: R
- **설명**:
  - 작업자가 임의로 자동 분류 공정을 즉시 일시정지할 수 있다.
  - 재가동 시 정지 전 공정 현황을 유지한 채 재개한다.

### S-18 자동 일시 정지
- **우선순위**: R
- **설명**: 자동 분류 공정 중 다음 상황이면 자동으로 일시정지한다.
  - 분류 배출 슈트에 물품이 가득 찬 경우
  - 물품이 자동 분류 공정의 특정 구간에 끼인 경우

### S-19 경고 알림
- **우선순위**: R
- **설명**: 다음 상황에서 관리자와 작업자에게 알림을 보낸다.
  - '미분류 판정' 이벤트 발생 시
  - '실시간 분류 오류 감지' 이벤트 발생 시
  - '일시 정지' 이벤트 발생 시

## 7. 실시간 모니터링

### S-20 관리자 실시간 모니터링
- **우선순위**: R
- **설명**: 관리자가 자동 분류 공정 현황을 실시간으로 파악할 수 있다.
- **표시 내용**:
  - 물품 분류 현황
  - 전체 분류 작업량 현황(갯수, 브랜드/간장종류별 수량)
  - 물품별 분류 현황(갯수, 브랜드/간장종류별 수량)

### S-21 작업자 실시간 모니터링
- **우선순위**: R
- **설명**: 작업자가 자동 분류 공정 현황을 실시간으로 파악할 수 있다.
- **표시 내용**:
  - 물품 분류 현황
  - 전체 분류 작업량 현황(갯수, 브랜드/간장종류별 수량)
  - 물품별 분류 현황(갯수, 브랜드/간장종류별 수량)
