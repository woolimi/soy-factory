# 한글 PDF 폰트 (선택)

`inbound_invoice_pdf.py`에서 한글이 깨지면 이 폴더에 한글 TTF 폰트를 넣어주세요.

- **NanumGothic.ttf** — [나눔고딕](https://github.com/naver/nanumsquare/releases) 등에서 받아 두면 됩니다.
- **Malgun.ttf** — Windows 맑은 고딕(시스템에 있으면 스크립트가 자동으로 사용).

macOS/Linux에서는 시스템에 설치된 한글 폰트(AppleGothic, Nanum, Noto 등)를 자동으로 찾습니다.
